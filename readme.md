# GameRpg — Mobile Client (MAUI)

Мобильное приложение для RPG-игры, написанное на **.NET MAUI**. Является клиентской частью проекта и взаимодействует с [GameRpg ASP.NET API](https://github.com/makksesh/GameApi). Поддерживает iOS и Android.

---

## Функциональность

- **Аутентификация** — регистрация, вход, смена пароля, автовосстановление сессии при запуске
- **Персонаж** — создание персонажа, просмотр характеристик, прокачка статов (HP, Mana, Armor, Damage)
- **Инвентарь** — просмотр предметов, экипировка/снятие предметов
- **Навыки** — просмотр всех навыков, изучение новых, повышение уровня навыков
- **Торговля** — просмотр активных лотов, создание лота, покупка, отмена лота
- **Друзья** — список друзей, отправка/принятие/отклонение заявок в друзья
- **Поддержка** — создание тикетов, просмотр своих обращений, закрытие/переоткрытие
- **Профиль** — просмотр данных аккаунта, выход из системы
- **Админ-панель** — управление пользователями (блокировка, разблокировка, смена роли) — доступно только для роли `Admin`
- **Модератор** — поддерка для пользователей — доступно только для роли `Moderator`

## Настройка базового адреса API

В `ServiceCollectionExtensions.cs` адрес API выбирается по платформе:

```csharp
#if IOS
    client.BaseAddress = new Uri("http://localhost:7001/api");
#elif ANDROID
    client.BaseAddress = new Uri("http://10.0.2.2:7001/api");
#else
    client.BaseAddress = new Uri("http://localhost:7001/api");
#endif
```

# Клонировать репозиторий
```
git clone <repo-url>
cd MobileApp
```
# Восстановить зависимости
```
dotnet restore
```
# Запустить на iOS-симуляторе (macOS + Xcode)
```
dotnet build -t:Run -f net10.0-ios
```
# Запустить на Android-эмуляторе
```
dotnet build -t:Run -f net10.0-android
```
